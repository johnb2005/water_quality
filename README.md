# Water Quality Dashboard

An interactive **Streamlit dashboard** for exploring water quality data from **Biscayne Bay**, featuring descriptive statistics, 2D and 3D visualizations, and integration with **NASA’s Astronomy Picture of the Day (APOD)**.  

This project demonstrates **data visualization, API integration, environment variable management, and clean project structure** — all skills relevant for internships.

---

## Features

- **Descriptive Statistics**
  - View raw water quality data
  - Automatically generated summary statistics with Pandas  

- **2D Visualizations (Plotly)**
  - Line chart of temperature over time
  - Scatter plots showing relationships between dissolved oxygen, temperature, and pH  

- **3D Visualization**
  - Interactive 3D scatter plot of geographic coordinates and water depth
  - Color-coded by temperature  

- **NASA APOD Integration**
  - Fetches NASA’s Astronomy Picture of the Day
  - Displays the image and explanation dynamically
